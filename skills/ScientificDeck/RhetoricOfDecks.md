# Rhetoric of Decks — Design Principles

> Source: Scott Cunningham (Baylor University). Adapted for PAI.

## Three Foundational Laws

1. **Beauty = Clarity** — Every visual element must justify its presence. Decoration without function is noise.
2. **Cognitive Load is the Enemy** — Minimize working memory burden. One idea per slide. No competing elements.
3. **Slides Support Speech** — Slides are visual anchors for spoken argument, not standalone documents.

## Aristotelian Rhetoric Framework

| Mode | Purpose | Weight (Academic Seminar) | Weight (Product Pitch) |
|------|---------|--------------------------|----------------------|
| **Ethos** (credibility) | Demonstrate competence, acknowledge limitations | 20% | 25% |
| **Pathos** (emotion) | Connect to audience aspirations and concerns | 10% | 40% |
| **Logos** (logic) | Present clear evidence and reasoning | 70% | 35% |

## One Idea Per Slide Rule

Every slide communicates exactly ONE claim. Test: can you state the slide's message in one sentence? If not, split it.

## Assertive Titles

Titles are claims, not labels.

| Bad (Label) | Good (Claim) |
|-------------|-------------|
| "Results" | "Treatment increased distance by 61 miles" |
| "Data Overview" | "Sample covers 12,000 firms across 8 years" |
| "Methodology" | "Difference-in-differences isolates causal effect" |
| "Summary" | "Three mechanisms explain the full treatment effect" |

## Marginal Analysis for Information Density

Every element added to a slide has a marginal benefit (MB) and marginal cost (MC = cognitive load).
- If MB > MC → add the element
- If MB < MC → remove it
- Distribute cognitive density evenly across the deck (no dense-then-sparse patterns)

## Narrative Arc Patterns

| Pattern | Structure | Best For |
|---------|-----------|----------|
| **Problem → Investigation → Resolution** | Classic academic | Research seminars |
| **Question → Evidence → Answer** | Socratic | Teaching |
| **Status Quo → Disruption → New Equilibrium** | Change narrative | Policy presentations |
| **Claim → Evidence → Implications** | Deductive | Conference talks |

## Audience Calibration

- **External (seminars/conferences):** Higher production value, more context, assertive titles, polished figures
- **Internal (coauthors):** Can be rougher, focus on methodology details, show more code/data
- **Students:** More pathos, build intuition before formalism, use concrete examples
