# Quality Checklist — HTML Scientific Decks

## Content
- [ ] Every slide: one main idea
- [ ] All titles: assertive claims
- [ ] Narrative arc coherent
- [ ] Cognitive load balanced
- [ ] Sources cited on data slides
- [ ] Slide count ~ talk minutes

## Technical
- [ ] KaTeX equations render (no raw LaTeX)
- [ ] Charts have labels/legends
- [ ] Mermaid diagrams render
- [ ] No broken images
- [ ] No text overflow at 100dvh

## Visual
- [ ] WCAG AA contrast (4.5:1)
- [ ] Dark + light mode work
- [ ] Responsive: 768px, 1024px, 1920px
- [ ] No AI slop (no Inter, no violet, no emoji headers)

## Navigation
- [ ] Keyboard works (arrows, space)
- [ ] Progress bar updates
- [ ] Scroll-snap aligns
